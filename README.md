PhotoView
===========================
###用 node.js、mongoDB、bootstrap 實作一個有 RWD 效果的相片檢視網頁
----------

建置環境:
----------
###1. 安裝 node.js

###2. 安裝 mongoDB

###3. 輸入指令 npm install 安裝所需 package

###4. 輸入指令
* 若沒有開啟本地端 mongoDB ， 則輸入指令 node app 執行程式 ， 
* 若有開啟則輸入指令 node app 'port編號' 執行程式 (ex: node app 27017) 

###5. 網址列輸入 http://127.0.0.1:3000/
